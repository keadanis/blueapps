Project Task-Subtask Merge Odoo Apps

Project Management Task merge Odoo apps helps to merge two or more project task into single task and when this task is merge it also merge timsheet of all tasks.

Features

Merge Task of same project

You can merge current task information to other task of same project.

Merge value of current task

Merge value of current task(Description, Timesheets,Planned Hours and Tags).
Demo Request Video Tutorial

Project Task Merge

On task form view there is a "Task Merge" button for merge current task to other task of same project.


Task Merge Wizard

In this wizard you need to select task from same project, Assigned to Deadline and Project is set according to current task.


Display merge task after merge with all value of current task(Description, Timesheets,Planned Hours and Tags).

You can see a message in chatter in merged task.

This message shows source document of new merged task.
